### php变量数据类型

#### 变量

- 什么是变量
    - 变量是用于存储信息的"容器"
- 变量的命名规则
    - 变量名以 $ 开头
    - 变量名由 字母 数字 下划线 组成
    - 变量名不能以数字开头

#### 数据类型

- 四种标量类型
    - 布尔型 bool
        - true
        - false
    - 整型   int
    - 浮点型 float
    - 字符型 string
        - 单引号不可以解析特殊字符和变量，双引号则可以

- 两种复合结构
    - 数组 array
    - 对象 object
        - new obj

- 两种特殊类型
    - null 空
    - resource 资源
