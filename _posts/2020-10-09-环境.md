#### 环境就是条件。

php环境即编译php所需要的所有条件

每一种语言都是一种约定。不是乱来写都能执行的，有约定就要有东西去判断是否合规则。

合规则然后怎么去执行。这就是PHP环境。

- 服务器
    - apache: web 服务器软件，负责回应所有收到的Web请求
    - Nginx:是一个高性能的HTTP和反向代理服务器，也是一个IMAP/POP3/SMTP服务器
- 数据库
    - mysql
- 操作系统
    - Windows
    - Linux
- 架构 主流php环境架构：
    - LAMP 即Linux + apache + mysql + php
    - WAMP 即Windows + apache + mysql + php
